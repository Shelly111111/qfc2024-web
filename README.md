## 功能

```
- Dashboard
  - 日志分析
  - 代码行数统计
  - 文本解密
  - Shell模拟
- 异常
  - 404 找不到
  - 500 服务器出错
- 帐户
  - 登录
  - 退出登录
```

## 使用

安装依赖：

```shell
npm install
```

运行：

```shell
npm start
```
